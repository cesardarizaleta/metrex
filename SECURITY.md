# Security Policy

We take security seriously. If you discover a vulnerability in Metrex:

- Please email security reports to security@nowhere.invalid with details and a minimal reproduction.
- Do not open a public issue for potential vulnerabilities.
- We will acknowledge receipt within 48 hours and provide a timeline for a fix when possible.

Supported versions:

- We support the latest minor release line published to npm.
- Security fixes may be backported at our discretion.

Thank you for responsibly disclosing issues and helping keep the community safe.
